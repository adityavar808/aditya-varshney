# Major Projects

## InterviewVerse AI

An AI-powered interview preparation platform that simulates real technical and HR interviews using Large Language Models. It provides resume-based question generation, AI interview sessions, performance evaluation, personalized feedback, and an interactive dashboard to help users improve their interview skills.

**Technologies:** React.js, Node.js, Express.js, MongoDB, Python, LLM APIs.

---

## Real-Time Video Analysis

A computer vision application capable of processing live video streams to detect, classify, and analyze objects in real time. The project leverages deep learning models and OpenCV for efficient video processing and AI-based visual understanding.

**Technologies:** Python, OpenCV, PyTorch, TensorFlow, Flask.

---

## Zenix

Zenix is an AI-powered personal assistant and productivity platform designed to simplify daily workflows using conversational AI. It integrates intelligent chat capabilities, automation features, and modern web technologies to provide users with a seamless digital assistant experience.

**Technologies:** React.js, Node.js, Express.js, MongoDB, Python, LLM APIs.
